# #1b SkeletonComposer Rubric

## 항목

1. **법령 강제 섹션 누락 (0.3, hard fail)** — 외국인 고용 계획서 같이 법령이 요구하는 섹션이 누락됐는가? 누락 시 강제 0점
2. **섹션 순서 (0.2)** — 행정 관행 순서를 따르는가? (배경/사유 → 효과 → 계획 → 기타)
3. **role·logic_anchor 명확성 (0.2)** — 각 섹션이 답해야 할 질문이 명시되었는가?
4. **소스 우선순위 적용 (0.2)** — `composition_meta.primary_source.kind`가 명세대로 적용됐는가? (시드 적중 시 official_form 또는 user_library 우선)
5. **불필요한 섹션 (0.1)** — 행정사 기준에서 군더더기 섹션이 끼어있지 않은가?

## 감점

- 법령 강제 섹션 누락: 0점
- 5섹션 미만 (외국인 고용·내용증명): -0.3
- role이 빈 문자열: -0.2
