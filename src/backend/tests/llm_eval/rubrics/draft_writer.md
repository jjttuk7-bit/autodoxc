# #6 DraftWriter Rubric

## 항목

1. **행정 톤 (0.2)** — 경어체·객관체 일관성. 1인칭·구어체·감정어 사용 시 감점
2. **법령 인용 형식 (0.15)** — "○○법 제○조 제○항" 표준 형식 준수
3. **사실 정확성 (0.2)** — 입력 facts와 본문이 일치하는가? (할루시네이션 X)
4. **annotation 일관성 (0.15)** — 각 문단의 `status`가 본문 내용과 일치하는가? (확정 정보를 inferred로 표시하면 감점)
5. **자리표시자 컨벤션 (0.1)** — `[[필드명]]` 형식. 다른 표기 시 감점
6. **빈 슬롯 추적 (0.1)** — empty 문단이 `empty_slots`에도 등록되었는가?
7. **금지 표현 (0.1)** — 1인칭(저희가), 감정어, 비공식 어미가 없는가?

## 감점

- 1인칭/감정어 1회: -0.1, 반복: -0.3
- 사실과 다른 수치: -0.5
- annotation status 불일치: -0.2 per case
